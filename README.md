# IIF-Global-Macro-Views---Europes-second-China-shock
China’s weak domestic demand and industrial upgrading are pushing more high-end output into Europe. The report argues that Europe’s “second China shock” is centered on technological convergence in autos, machinery, and clean tech, with implications for trade policy, industrial strategy, and investment screening.
Author: Marcello Estevão, Ugras Ulku, Gene Ma, Phoebe Feng, Leila Hamilton, Meshal Alahmadi, Runshi Xue
Date: March 3, 2026
